# tp5.1+layui通用后台模板

#### Description
用于快速搭建后台,通用crud 易于管理


#### Installation

1. 安装数据库文件 admin.sql
2. 修改database.php 库名 密码 等


#### 使用包

1. 百度富文本编辑器
2. [easypay](https://gitee.com/yansongda/pay) 微信阿里支付
3. [layui](https://www.layuion.com/doc/)后台模板
4. 核心框架[THINKPHP](https://www.kancloud.cn/manual/thinkphp5_1) 5.1
